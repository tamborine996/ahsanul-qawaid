# Ahsanul Qawaid - Interactive Arabic Learning

An interactive web app for children learning to read Arabic in preparation for Quran recitation.

**Live App:** https://tamborine996.github.io/ahsanul-qawaid/

## Overview

Based on the Ahsanul Qawaid methodology, this app covers the complete curriculum (Lessons 3-30):
- Basic vowel marks (Fatha, Kasra, Damma)
- Tanween (Fathatain, Kasratain, Dammatain)
- Madd letters (stretching with Alif, Waw, Ya)
- Standing vowels
- Sukoon and Leen letters
- Silent letters and Hamzah
- Shaddah (doubling)
- Advanced Madd rules
- Quranic phrases and practice
- Noon Sakin rules
- Waqf (stopping marks)
- Huroof Muqatta'at

## Features

- **28 Lessons** - Complete Ahsanul Qawaid curriculum
- **Tap-to-reveal transliteration** - Hidden by default to encourage learning Arabic script
- **Long-press to mark** items for practice (coral highlight)
- **Practice Mode** - Fullscreen cycling through marked items
- **Shuffle mode** - Randomize exercise order
- **Chunked exercises** - 10 items per page with navigation
- **Child-friendly tips** - Simple explanations for each lesson
- **Progress tracking** - Completed lessons saved automatically
- **Mobile-friendly** - Works on phones, tablets, and computers

## How to Use

1. **Open the app** at https://tamborine996.github.io/ahsanul-qawaid/
2. **Select a lesson** from the navigation buttons
3. **Tap any card** to reveal the pronunciation (auto-hides after 2 seconds)
4. **Long-press (hold)** any card to mark it for practice
5. **Click "Practice"** button to cycle through marked items
6. **Click "Shuffle"** to randomize the exercise order
7. **Mark as Complete** when finished with a lesson

## Running Locally

```bash
cd Qaida
npx serve
```

Then open http://localhost:3000
