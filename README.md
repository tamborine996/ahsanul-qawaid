# Ahsanul Qawaid - Arabic Letters

An interactive web app for children learning to read Arabic in preparation for Quran recitation.

## Overview

Based on the Ahsanul Qawaid methodology, this app teaches:
- The 28 Arabic letters
- Basic vowel marks (harakat): Fatha, Kasra, Damma
- Tanween (double vowels): Fathatain, Kasratain, Dammatain

## Features

- **7 Lessons** covering letters and vowel marks
- **Right-to-left layout** for authentic Arabic reading direction
- **Long-press to mark** items that need practice (visual bookmark for parent-guided learning)
- **Practice counter** shows items marked, click to jump to first one
- **Mobile-friendly** responsive design

## Running the App

```bash
cd Qaida
npx serve
```

Then open http://localhost:3000

## Lessons

1. **Letters** - All 28 Arabic letters
2. **Fatha** - The "a" sound
3. **Kasra** - The "i" sound
4. **Damma** - The "u" sound
5. **Fathatain** - The "an" sound (tanween)
6. **Kasratain** - The "in" sound (tanween)
7. **Dammatain** - The "un" sound (tanween)
